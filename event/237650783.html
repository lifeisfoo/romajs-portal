LambCI: build your own TravisCI with AWS Lambda

15/2/2017 - 18:00:00

AWS Lambda is a new cloud paradigm in the Amazon cloud universe that enables you to use and pay exactly what you need, for the exact amount of time of use, leaving to Amazon all the hard part of orchestrating the instances. In this presentation we'll explore this new feature building a continuos integration system with LambCI for a team of small/medium size with ease in a cost effective way. This solution has been adopted in a real life for the past 4 months (so expect a lot of feedback)

Speaker: Marco Liberati (@dej611)
