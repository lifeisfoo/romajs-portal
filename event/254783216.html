Behavioral Programming with Redux: request, wait and block

#BehavioralProgramming is a new programming paradigm specifically suited for event-based applications such as UIs, and is aligned with how humans think. In this talk I will explain what problems it solves and how it can be used in the context of Redux and React.

Language: english Speaker: Luca Matteis

Unstated: state so simple

Quando l'applicazione React comincia ad assumere una certa complessità, tra operazioni asincrone e scambi con il server, uno state management è fondamentale per mantenere separata la logica applicativa dalla rappresentazione dei dati. Tra le molteplici soluzioni, Redux è tra le soluzioni più utilizzate e complete in tutte le sue varianti, ma la sua complessità iniziale nella definizione dello store, reducers, action, saga, epic, etc.. a volte può essere un overkill per applicazioni di piccola grandezza. Unstated, basato sulle nuove context API, permette una gestione dello stato in maniera molto più semplice.

Language: italian Speaker: Matteo Manchi
