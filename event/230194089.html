Interface design with React.js

20/4/2016 - 17:00:00

http://photos2.meetupstatic.com/photos/event/6/d/c/4/600_448648100.jpeg

React is a library, but it's not just a library. It brings about a new way of thinking and builds user interfaces. During this presentation we will talk about what is React and how it can help us to construct UI writing Components.We'll see how React could be used to create a responsive layout, to organize our code with js-routing and how to personalize our components with inline-style.

The meeting will be opened by the tip of the month: "Functional Programming: Like Erlang but in JavaScript"
