Il web è per tutti? Il tema dell'accessibilità

19/4/2017 - 17:00:00

"Il web è per tutti" è un motto che abbiamo sentito pronunciare innumerevoli volte negli ultimi decenni. Ma nella realtà è facile incappare in siti ed applicazioni web di difficile comprensione ed utilizzo da parte di chi ha esigenze speciali o naviga il web con strumenti diversi dal solito browser. In questo talk introdurremo la tematica dell'accessibilità web, illustrandone le motivazioni , gli standard tecnici di riferimento e alcuni strumenti e metodologie che potranno aiutare gli sviluppatori a rendere il web davvero "un luogo per tutti".

Speaker: Antonio Giovanni Schiavone
