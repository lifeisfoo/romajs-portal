Statecharts: Controlling the behavior of complex systems in a visual way

Statecharts is a visual formalism created in the 80s that allows us to model complex behavior in a visual way. In this talk I will explain what problems it solves, and more importantly I will show examples on how it can be used within real-world applications.

Language: English

Speaker: Luca Matteis (@lmatteis)

Write once, debug everywhere

JS is a beautiful language and can be deployed to the web where it works on any browser. But what happens when it doesn’t? Debugging Javascript applications can be very hard, especially when diving into CSS, asynchronous flows or complex logics. In this talk we’re going to explore the world of browser Dev Tools and show how each browser shines at specific features compare to others.

Language: Italian

Speakers: Marco Liberati (@dej611) & Gabriele Falasca (@gabrycaos)
