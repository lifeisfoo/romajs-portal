Costruire una Dashboard interattiva + Intro a Firebase DB - RomaJS #AperiTech

21/11/2018 - 18:00:00

IMPORTANTE: per accedere facilmente agli spazi offerti da EnLabs è necessario registrare la propria presenza su: https://www.eventbrite.it/e/biglietti-costruire-una-dashboard-interattiva-intro-a-firebase-db-romajs-aperitech-50534605365

Costruire una pipeline efficiente per una dashboard: GraphDB, GraphQL, React e d3 In questo talk parleremo di come costruire una dashboard interattiva con React e d3 sfruttando una pipeline efficiente con GraphQL ed un Graph Database: l’obiettivo finale è quello di costruire visualizzazioni personalizzate sfruttando la flessibilità di GraphQL e l’efficienza di un Graph Database

Speaker: Marco Liberati, Enrico Risa

Presentazione del Firebase Realtime Database

Firebase è la piattaforma Google dedicata allo sviluppo mobile, ma non solo.. Tra tutti i servizi disponibili sulla piattaforma, vedremo in dettaglio il Realtime Database che gestisce dati in formato JSON, le API JavaScript per la lettura e scrittura dei dati e daremo uno sguardo alle Rules per gestire i dati in sicurezza.

Speaker: Riccardo Berti
