GraphQL App with Node.js and Apollo

16/11/2016 - 18:00:00

http://photos4.meetupstatic.com/photos/event/f/2/0/600_456063872.jpeg

GraphQL is a data query language for backend developed at Facebook and designed to be an alternative to REST. In this talk after an introdution to GraphQL we'll see how to create a GraphQL server using Node.js with Apollo Server, and how to use those APIs from a React client.

Speaker: Enrico Risa (@wolf4ood)

Costruire una Applicazione GraphQL con Node.js e Apollo

GraphQL è un linguaggio di query per backend sviluppato da Facebook come alternativa all'architettura REST. In questa sessione, dopo una introduzione a GraphQL, vedremo come costruire un server GraphQL usando Node.js con Apollo Server e come usare queste APIs da un client React.

Relatore: Enrico Risa (@wolf4ood)
