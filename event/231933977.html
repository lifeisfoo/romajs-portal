Cycle.js - A functional and reactive JavaScript framework for cleaner code

21/6/2016 - 17:00:00

Cycle.js is a new and very unusual JavaScript framework that uses Observables (RxJS) for developing applications. Luca Matteis will discuss the 3 main concepts behind Cycle.js, and how reactive and functional programming might be the next generation paradigm for developing UIs.

The meeting will be opened by the tip of the month: No-sense JavaScript operations - Matteo Manchi

====

Cycle.js è un framework JavaScript che usa il concetto di Observables (RxJS) per lo sviluppo di applicazioni. Luca Matteis ci illustrerà i 3 concetti principali dietro Cycle.js, e come reactive and functional programming potrebbero essere il nuovo paradigma per lo sviluppo di interfacce.

L'incontro verrà aperto dalla pillola del mese: No-sense JavaScript operations - Matteo Manchi
