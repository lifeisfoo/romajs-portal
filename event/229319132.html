Introduction to Machine Learning - Piero Savastano

16/3/2016 - 18:00:00

"Machine learning" is applied Artificial Intelligence, a family of algorithms able to learn and generalize on data in order to automate tasks. Machine learning researchers and web developers are collaborating to build open source libraries for "smart" applications. This presentation will introduce to the basic concepts and applications of machine learning. In the context of Javascript we will overview available libraries and a set of examples on the manipulation of textual data.

The meeting will be opened by the tip of the month: Objects, prototype e classes from ES5 to ES6 - Claudio Mantuano

====

Il "machine learning" è l'intelligenza artificiale applicata, una famiglia di algoritmi che apprendono dai dati al fine di automatizzare le decisioni. La collaborazione tra ricercatori di machine learning e sviluppatori web ha portato alla nascita di librerie open source per rendere le applicazioni più "intelligenti". La presentazione introduce ai concetti di base del machine learning e agli utilizzi più concreti. Nello specifico di Javascript faremo una panoramica delle librerie open e degli esempi pratici su come maneggiare dati testuali.

L'incontro verrà aperto dalla pillola del mese: Oggetti, prototype e classi viaggio da ES5 a ES6 - Claudio Mantuano
